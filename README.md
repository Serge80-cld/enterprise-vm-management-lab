# Enterprise VM & EC2 Management Lab

Lab avancé simulant un environnement entreprise pour la création, la sécurisation et la maintenance de VM (Azure) et d'instances EC2 (AWS) via Terraform et PowerShell.

## Objectifs

- Démontrer la maîtrise d'Infrastructure as Code (Terraform) sur Azure et AWS
- Mettre en place des bonnes pratiques de sécurité, de maintenance et de FinOps
- Documenter des scénarios d'exploitation et d'incidents comme en environnement grand compte

## Structure

- `docs/` : documentation, architecture, sécurité, FinOps, runbooks
- `terraform/azure` : code Terraform pour les VM Azure
- `terraform/aws` : code Terraform pour les instances EC2 (mode simulation)
- `powershell/` : scripts d'automatisation (maintenance, patching, monitoring, backups)
